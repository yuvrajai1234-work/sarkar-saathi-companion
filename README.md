🌾 Sarkar Saathi — AI Companion for Government Schemes Sarkar Saathi is an AI-powered multilingual voice assistant designed to help rural citizens access government welfare schemes easily through voice— without needing smartphones, internet literacy, or complex paperwork knowledge.

🚀 Problem Statement Millions of rural citizens struggle to access welfare programs due to: Language barriers Low digital literacy Complex eligibility rules Poor internet connectivity Lack of awareness about available schemes Sarkar Saathi bridges this gap with a voice-first AI access layer for public services.

💡 Solution Overview Sarkar Saathi enables citizens to: Speak in their native language Discover eligible government schemes Get step-by-step guidance Receive SMS updates & document checklists Apply without navigating portals All through phone calls, WhatsApp, SMS, or USSD.

✨ Key Features Multilingual Voice Interface Supports Hindi, Tamil, English Voice-First Design No typing or reading required — fully conversational. Smart Eligibility Engine AI asks targeted questions and validates eligibility in real time. Scheme Database Covers PM-KISAN, Ayushman Bharat, MGNREGA, and 50+ schemes. Offline Capability Works via SMS & USSD on 2G networks. SMS Notifications Sends application links, document lists, and status updates. Integration Ready Connects with DigiLocker, Aadhaar, and UMANG.

🧭 User Journey Connect User calls or messages Sarkar Saathi via phone/SMS/WhatsApp. Understand Needs AI asks what support the user is looking for. Eligibility Check Collects details like income, land, Aadhaar, bank info. Guidance & Support Provides next steps, documents, and application tracking.
